## 사석훈
게임 이론이라는 카테고리를 조금 더 이해 할 수 있는 계기가 되었다. 코드로 구현 할 만한 수준의 게임 이론은 현상 에 관하여 규칙을 찾는 개념 인 것 같다
## 지용주
몇가지 케이스를 계산해본 후 패턴을 찾아서 문제를 해결하였다. 패턴만 찾으면 코드 자체는 어렵지 않았다.
## 최이승우
친구 말대로 작은 수부터 직접 계산해 반복되는 패턴을 찾고, 그 규칙을 일반화해 답을 예측하는 문제였다.
