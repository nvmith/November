## 박강현
- 숫자를 더해가는 방식

처음에는 인덱스를 감소시키며 조건을 설정해 차례대로 감소하는 수를 구하려고 했지만, 구현이 복잡하고 비효율적이라는 판단이 들어 방향을 바꾸게 되었다. <br>
결국 감소하는 수의 최댓값이 9876543210이라는 점에 착안하여, 숫자를 정해두고 그보다 작은 수를 뒤에 붙이는 방식으로 전체 감소하는 수를 미리 생성한 뒤 정렬하면 문제를 효율적으로 해결할 수 있다고 생각했다. <br>
이러한 방식으로 접근하면서 문제를 성공적으로 해결할 수 있었다.
